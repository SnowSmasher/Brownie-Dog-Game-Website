Process

The process for making the map and the characters is relatively simple. First, we draw the character or map on an A4 sheet of paper, using pencil. After this, we scan the paper using a printer in order for turn it into a jpg that we can edit with a computer. We then use Adobe Photoshop to trace the drawing and then add colour to the drawing. This drawing can then be used in our game through Unity, the game engine we are using for this project.
